MetaLink is a platform for sharing metaverse world as weblink

## Frontend

The frontend codes are located on `./frontend` dir

### requirements
- node v14.16^
- React
- next.js
- styled-components

### Setup
```
yarn install
yarn dev
```

### Build
```
yarn build
// then created `.next` folder so you can serving to anywhere you want to do
```
